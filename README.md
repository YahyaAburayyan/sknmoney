# SknMoney — Shared Expense Tracker

**Live → [sknmoney.vercel.app](https://sknmoney.vercel.app)**

Track shared expenses with roommates. Everyone sees who owes what, in real time.

---

## Features

- Log expenses — who paid, how much, and who it's split between
- Equal or custom splits
- Real-time balance tracking
- Debt simplification — calculates the minimum transactions to settle up
- Mark as Paid when money changes hands
- Arabic / English with full RTL support
- Invite roommates via 6-character code

---

## Stack

Next.js 15 · TypeScript · Supabase (Postgres + Auth + Realtime) · Tailwind CSS · Vercel
